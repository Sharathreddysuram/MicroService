# Banking Microservices Project

This project is a complete Banking Application built using Spring Boot Microservices Architecture.

## Architecture

The project contains the following services:

- Service Registry (Eureka Server)
- Config Server
- API Gateway
- Customer Service
- Account Service
- Auth Service
- Transaction Service

## Technologies Used

- Java 23
- Spring Boot
- Spring Cloud
- Eureka Server
- Spring Cloud Config
- Spring Cloud Gateway
- OpenFeign
- Spring Data JPA
- MySQL
- Maven
- Git & GitHub

---

## Services and Ports

| Service | Port |
|---|---|
| Service Registry | 8761 |
| Config Server | 8888 |
| API Gateway | 8084 |
| Customer Service | 8081 |
| Account Service | 8083 |
| Auth Service | 8085 |
| Transaction Service | 8086 |

---

## Features

- Centralized Configuration using Config Server
- Service Discovery using Eureka
- API Routing using Gateway
- Inter-Service Communication using OpenFeign
- Database Integration using MySQL
- Microservices-based Architecture
- Scalable and Modular Design
